Essential Unix Commands for Developers

Master the command line to boost your productivity

Unix commands are essential tools for developers to navigate and manipulate files, manage processes, and automate tasks. Here's a comprehensive guide to the most important commands every developer should know.

File System Navigation

pwd - Print Working Directory

pwd

Displays the full path of the current working directory.

ls - List Directory Contents

ls [options] [directory]

Lists files and directories in the current directory.

Common options:

-l Long format (detailed listing)

-a Show hidden files

-h Human-readable file sizes

cd - Change Directory

cd [directory]

Changes the current working directory.

Examples:

cd /path/to/directory - Change to absolute path

cd .. - Move up one directory

cd ~ - Go to home directory

cd - - Go to previous directory

File Operations

cp - Copy Files/Directories

cp [options] source destination

Copies files and directories.

Examples:

cp file.txt /backup/ - Copy file to directory

cp -r dir1/ dir2/ - Copy directory recursively

mv - Move/Rename Files

mv [options] source destination

Moves or renames files and directories.

Examples:

mv oldname.txt newname.txt - Rename file

mv file.txt /new/location/ - Move file

rm - Remove Files/Directories

rm [options] file/directory

Removes files and directories.

Warning: Deleted files cannot be recovered. Use with caution!
Common options:

-r Recursive (for directories)

-f Force (no confirmation)

-i Interactive (ask for confirmation)

File Viewing and Editing

cat - Concatenate and Display

cat [file]

Displays file contents.

less/more - File Pager

less [file]

Views file contents one screen at a time.

Navigation in less:

Space - Next page

b - Previous page

q - Quit

/pattern - Search forward

?pattern - Search backward

head/tail - View File Start/End

head -n 10 file.txt # Show first 10 lines tail -n 20 file.log # Show last 20 lines

grep - Search Text

grep [options] pattern [file...]

Searches for patterns in files.

Examples:

grep "error" log.txt - Search for "error" in file

grep -r "function" src/ - Recursive search

grep -i "warning" file.txt - Case-insensitive search

File Permissions

chmod - Change File Mode

chmod [options] mode file

Changes file permissions.

Examples:

chmod +x script.sh - Make file executable

chmod 644 file.txt - rw-r--r--

chmod -R 755 directory/ - Recursively change permissions

chown - Change Owner

chown [options] user:group file

Changes file owner and group.

Process Management

ps - Process Status

ps aux

Lists all running processes.

top/htop - Process Viewer

top

Interactive process viewer (q to quit).

kill - Terminate Process

kill [signal] PID

Sends a signal to a process.

Common signals:

kill -9 1234 - Force kill process 1234

kill -15 1234 - Graceful termination

Networking

ping - Test Network Connectivity

ping [hostname/IP]

curl - Transfer Data

curl [options] [URL...]

Transfers data to/from a server.

Examples:

curl -O https://example.com/file.zip - Download file

curl -X POST -d "data=example" http://api.example.com - Send POST request

ssh - Secure Shell

ssh user@hostname

Connects to a remote server securely.

scp - Secure Copy

scp source user@host:destination

Copies files between hosts securely.

System Information

df - Disk Free

df -h

Shows disk space usage.

du - Disk Usage

du -sh [directory]

Shows directory space usage.

free - Memory Usage

free -h

Shows memory usage.

uname - System Information

uname -a

Shows system information.

Package Management (Ubuntu/Debian)

apt - Package Management

sudo apt update # Update package lists sudo apt upgrade # Upgrade all packages sudo apt install package # Install a package sudo apt remove package # Remove a package

Useful Shortcuts

  • Ctrl+C - Stop current command
  • Ctrl+Z - Suspend current command (bg/fg to manage)
  • Ctrl+D - End of file/Exit shell
  • Ctrl+R - Search command history
  • !! - Repeat last command
  • !$ - Last argument of previous command
  • command > file - Redirect output to file (overwrite)
  • command >> file - Append output to file
  • command1 | command2 - Pipe output of command1 to command2

Text Processing

sed - Stream Editor

sed [options] 's/pattern/replacement/' file

Performs text transformations on an input stream.

Examples:

sed 's/foo/bar/g' file.txt - Replace all 'foo' with 'bar'

sed -i.bak 's/old/new/' file.txt - Edit file in place with backup

awk - Pattern Scanning and Processing

awk 'pattern { action }' file

Powerful text processing and data extraction.

Examples:

awk '{print $1}' file.txt - Print first column

awk '/error/ {count++} END {print count}' log.txt - Count errors

sort - Sort Lines of Text

sort [options] [file]

Sorts lines in a file.

Common options:

-n - Numeric sort

-r - Reverse order

-u - Unique lines only

uniq - Report or Omit Repeated Lines

uniq [options] [input [output]]

Filters adjacent matching lines from input.

Example:

sort file.txt | uniq -c | sort -nr - Count and sort unique lines

Advanced File Operations

find - Search for Files

find [path] [expression]

Searches for files in a directory hierarchy.

Examples:

find /path -name "*.log" - Find files by name

find . -type f -mtime -7 - Files modified in last 7 days

find . -size +10M -exec ls -lh {}\; - Find large files

tar - Archive Files

tar [options] [archive-file] [files]

Creates and extracts archive files.

Common operations:

tar -czvf archive.tar.gz directory/ - Create gzipped archive

tar -xzvf archive.tar.gz - Extract gzipped archive

rsync - Remote File Sync

rsync [options] source destination

Efficiently copies and syncs files locally or remotely.

Example:

rsync -avz /local/dir/ user@remote:/remote/dir/ - Sync directories

System Monitoring and Performance

htop - Interactive Process Viewer

htop

Interactive process viewer (install with sudo apt install htop).

iostat - CPU and I/O Statistics

iostat [options] [interval] [count]

Reports CPU and I/O statistics.

netstat - Network Statistics

netstat -tuln

Displays network connections, routing tables, and interface statistics.

lsof - List Open Files

lsof [options]

Lists information about files opened by processes.

Examples:

lsof -i :80 - Find processes using port 80

lsof /path/to/file - Find processes using a file

User and Permission Management

sudo - Execute as Superuser

sudo command

Executes a command with superuser privileges.

su - Switch User

su - username

Switches to another user account.

passwd - Change Password

passwd [username]

Changes user password.

usermod - Modify User Account

usermod [options] username

Modifies a user account.

Compression and Archiving

gzip/gunzip - File Compression

gzip file.txt # Compress gunzip file.txt.gz # Decompress

Compresses or decompresses files.

zip/unzip - Package and Compress Files

zip archive.zip files... # Create zip unzip archive.zip # Extract zip

System Information (Advanced)

lscpu - CPU Information

lscpu

Displays information about the CPU architecture.

lsblk - List Block Devices

lsblk

Lists information about all available block devices.

journalctl - Query Systemd Journal

journalctl [options]

Views and analyzes systemd logs.

Examples:

journalctl -u nginx - View nginx service logs

journalctl --since "1 hour ago" - Recent logs

Tip: Use man command to read the manual for any command (e.g., man ls). Press 'q' to quit the manual.
