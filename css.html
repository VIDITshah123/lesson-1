CSS Basics

What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages.

Basic CSS Syntax

selector {
    property: value;
}

Common CSS Properties

  • color - Text color
  • background-color - Background color
  • font-size - Text size
  • margin - External spacing
  • padding - Internal spacing
  • border - Border styling
