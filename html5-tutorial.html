HTML5 Capabilities

A Comprehensive Guide to Modern Web Development

1. Semantic Elements

HTML5 introduced semantic elements that clearly describe their meaning to both the browser and the developer.

Key Semantic Elements:

  • <header> - Represents introductory content
  • <nav> - Defines navigation links
  • <section> - Defines a section in a document
  • <article> - Represents independent, self-contained content
  • <footer> - Defines a footer for a document or section
  • <aside> - Defines content aside from the page content
  • <figure> and <figcaption> - For figures and captions

Example:

<article>
    <header>
        <h1>Blog Post Title</h1>
        <p>Published on <time datetime="2025-07-05">July 5, 2025</time></p>
    </header>
    <section>
        <p>Blog post content goes here...</p>
    </section>
    <footer>
        <p>Posted in: Web Development</p>
    </footer>
</article>

2. Multimedia Support

HTML5 provides native support for audio and video without requiring plugins.

Video Element

<video width="320" height="240" controls>
    <source src="movie.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

Audio Element

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
</audio>

3. Canvas for Graphics

The <canvas> element is used to draw graphics via JavaScript.

<canvas id="myCanvas" width="200" height="100" style="border:1px solid #000000;"></canvas>
<script>
    var canvas = document.getElementById("myCanvas");
    var ctx = canvas.getContext("2d");
    ctx.fillStyle = "#FF0000";
    ctx.fillRect(0, 0, 150, 75);
</script>

4. Form Enhancements

HTML5 introduced new form input types and attributes for better user experience.

New Input Types:

  • email - For email addresses
  • url - For web addresses
  • number - For numerical input
  • date - For date selection
  • color - For color selection
  • range - For a slider control
  • search - For search fields

Example Form:

<form>
    <div>
        <label for="email">Email:</label>
        <input type="email" id="email" required>
    </div>
    <div>
        <label for="birthdate">Birth Date:</label>
        <input type="date" id="birthdate">
    </div>
    <div>
        <label for="volume">Volume:</label>
        <input type="range" id="volume" min="0" max="100">
    </div>
    <button type="submit">Submit</button>
</form>

5. Local Storage

HTML5 provides two new objects for storing data on the client:

  • localStorage - Stores data with no expiration date
  • sessionStorage - Stores data for one session

Example Usage:

// Store data
localStorage.setItem('username', 'JohnDoe');

// Retrieve data
let user = localStorage.getItem('username');

// Remove data
localStorage.removeItem('username');

// Clear all data
localStorage.clear();

6. Geolocation

HTML5 Geolocation lets you get the geographical position of a user.

if (navigator.geolocation) {
    navigator.geolocation.getCurrentPosition(showPosition);
} else {
    alert("Geolocation is not supported by this browser.");
}

function showPosition(position) {
    let latitude = position.coords.latitude;
    let longitude = position.coords.longitude;
    console.log(`Latitude: ${latitude}, Longitude: ${longitude}`);
}

7. Web Workers

Web workers run JavaScript in the background, independently of other scripts.

Main JavaScript:

// Check if Web Worker is supported
if (window.Worker) {
    // Create a new Web Worker
    const myWorker = new Worker('worker.js');
    
    // Send message to worker
    myWorker.postMessage('Hello Worker!');
    
    // Receive message from worker
    myWorker.onmessage = function(e) {
        console.log('Message received from worker:', e.data);
    };
}

worker.js:

// Listen for messages from the main thread
self.onmessage = function(e) {
    console.log('Message received in worker:', e.data);
    
    // Send message back to main thread
    self.postMessage('Hello from Worker!');
};

8. Drag and Drop

HTML5 provides native drag and drop functionality.

<div id="div1" ondrop="drop(event)" ondragover="allowDrop(event)">
    <p id="drag1" draggable="true" ondragstart="drag(event)">Drag me!</p>
</div>
<div id="div2" ondrop="drop(event)" ondragover="allowDrop(event)"></div>

<script>
function allowDrop(ev) {
    ev.preventDefault();
}

function drag(ev) {
    ev.dataTransfer.setData("text", ev.target.id);
}

function drop(ev) {
    ev.preventDefault();
    var data = ev.dataTransfer.getData("text");
    ev.target.appendChild(document.getElementById(data));
}
</script>
